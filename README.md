# 🚀 API Testing Interview Q&A App

A beautifully designed, interactive interview preparation app covering everything from API basics to expert-level concepts.

## 📚 Topics Covered (80 Questions)

| Section | Level | Topics |
|---|---|---|
| 01 · API Fundamentals | Beginner | REST, HTTP methods, Status codes, JSON, SOAP vs REST |
| 02 · API Testing Basics | Beginner | What to test, Postman, API vs UI testing |
| 03 · Authentication & Security | Intermediate | JWT, OAuth 2.0, OWASP API Top 10 |
| 04 · Tools & Automation | Intermediate | Postman, Newman, Python Requests, REST Assured |
| 05 · Advanced Concepts | Advanced | GraphQL, Rate Limiting, Versioning, Contract Testing, Mocking |
| 06 · Performance Testing | Advanced | Load, Stress, Spike, k6 |
| 07 · Senior & Expert Level | Expert | Framework design, Schema validation, Microservices, Chaos Engineering |

## ✨ Features

- 🎯 80 detailed interview questions with answers
- 🔍 Live search across all questions
- 🏷️ Filter by difficulty level (Beginner → Expert)
- 📊 Progress tracker (how many questions you've explored)
- 💻 Syntax-highlighted code examples
- 💡 Tip boxes and note boxes
- 📱 Mobile responsive design
- ↑ Back to top button
